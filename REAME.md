# Hướng dẫn sử dụng
## Cấu hình:
- Điền `APP_ID` và `APP_KID` ứng với nội dung được tạo trong jitsi
- Tải file secret key về vbaf đổi tên là `key.pk`
- Điền các nội dung cần thiết trong `firebase.json`

# Cài thư viện
Cài thư viện bằng composer:
```base
    composer install 
```
